# Design Language (NexaLedger)

**Purpose:** Define the visual/UX identity in a way that guides small refinements.

**Who uses it:** AI + human devs.

**Change frequency:** Low–Moderate.

## Core aesthetic
- Calm, breathable, premium, operationally comfortable
- Subtle hierarchy: readable, emotionally comfortable, not crowded
- Restraint in motion and interaction

## Design intent (what must be preserved)
- Consistent spacing rhythm
- Soft hierarchy and muted secondary text
- Premium surfaces: rounded corners, subtle borders/backdrop blur

## What to avoid
- Flashy/experimental visual trends
- Overly animated interactions
- Heavy shadow density and competing visual anchors
